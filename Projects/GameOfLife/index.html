About

A C++ implementation of Conway's Game of Life, developed as part of my Game Development coursework at Full Sail University. This project focuses on the core logic of cellular automata, grid management, and efficient state updates within a console-based environment.

The project demonstrates foundational programming skills including memory management, loop logic, and conditional rules applied to a classic computer science simulation.

My Contributions

  • Implemented all four standard Game of Life rules for cell survival and birth from scratch in C++
  • Built the grid data structure and state buffer system for efficient generation-to-generation updates
  • Designed the simulation loop with configurable speed controls for generation stepping
  • Handled edge cases for grid boundary conditions and cell neighbor counting
  • Applied clean memory management and modular code structure following academic requirements

Features

  • Grid Logic — Implementation of all four standard Game of Life rules for cell survival and birth
  • Dynamic Grid — Support for configurable grid dimensions
  • Simulation Loop — Automated generation updates with controllable speed
  • Neighbor Counting — Accurate 8-directional neighbor detection with boundary handling
  • C++ Fundamentals — Clean use of memory management, loops, and conditional logic
