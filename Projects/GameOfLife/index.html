Project Overview

A C++ implementation of Conway's Game of Life, developed as part of my Game Development coursework at Full Sail University. This project focuses on the core logic of cellular automata, grid management, and efficient state updates within a console-based or simple graphical environment.

Features

  • Grid Logic — Implementation of the four standard Game of Life rules for cell survival and birth.
  • Dynamic Grid — Support for configurable grid dimensions.
  • Simulation Loop — Automated generation updates with controllable speed.
  • C++ Fundamentals — Clean use of memory management, loops, and conditional logic.
