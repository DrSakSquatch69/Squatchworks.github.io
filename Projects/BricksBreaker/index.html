Bricks Breaker Arcade Classic

A solo academic project implementing classic arcade mechanics in C++, focused on collision math and object-oriented design.

Project Overview

Bricks Breaker is a faithful recreation of the classic brick-breaking formula. Built using the Windows API for rendering, the project served as a deep dive into 2D collision physics, paddle movement logic, and level state management.

The codebase emphasizes clean OOP principles and efficient memory management within a constrained environment.
