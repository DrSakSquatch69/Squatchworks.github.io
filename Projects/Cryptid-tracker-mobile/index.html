About the Project

Cryptid Tracker is a mobile utility application for logging and tracking cryptid sightings. Rooted in Jacob's personal passion for cryptozoology and fringe science, this app is designed to give enthusiasts a clean, purpose-built tool for documenting encounters, pinning locations, and reviewing community sighting data.

The project is focused on strong UX fundamentals, intuitive information architecture, and a clean mobile-first design system. Built in Unity with a mobile deployment target.

Planned Features

  • Sighting Logger — Document cryptid encounters with notes, location, date, and type classification.
  • Interactive Map — Pin-based map view showing logged sightings geographically.
  • Cryptid Database — Built-in reference for known cryptid types and reported behaviors.
  • Photo Attachment — Attach photos or sketches to sighting reports.
  • Community Feed — Browse and filter sightings submitted by other users.
  • Offline Support — Core logging features work without an internet connection.

Design Principles

  • Mobile-first UI layout optimized for one-handed use.
  • Dark theme with cryptid-themed visual language.
  • Clean information hierarchy making logging fast and frictionless.
  • Accessibility-conscious component design.
