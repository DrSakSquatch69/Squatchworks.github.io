# Cryptid Tracker — Mobile

> **Role:** Solo Developer  
> **Engine:** Unity (C#)  
> **Platform:** Mobile (iOS / Android)  
> **Status:** 🚧 In Development  

---

## About

Cryptid Tracker is a mobile game currently in active development. Players take on the role of a field investigator tracking and documenting cryptid sightings across a mysterious landscape. The project began with a full Game Design Document and scene flow architecture before any code was written, following a design-first development process.

This project is being built independently outside of coursework to demonstrate end-to-end mobile game development skills — from GDD and UI wireframes through Unity scene architecture and C# gameplay systems.

- ✅ Game Design Document (GDD) complete
- ✅ Scene flow and navigation architecture defined
- ✅ Core scene skeleton implemented in Unity
- 🔄 Main gameplay scene in active development
- ⏳ Full feature implementation ongoing

---

## Planned Features

- **Cryptid Field Journal** — in-game log for documenting discovered cryptids with descriptions and rarity ratings
- **Investigation Gameplay** — location-based clue gathering and evidence collection mechanics
- **Mobile-First UI** — touch-optimized interface designed for portrait and landscape orientations
- **Progression System** — unlock new cryptid types and investigation sites as the player advances
- **Save System** — persistent player progress across sessions
